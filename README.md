# ブックマーク中クリックアクティブ化 (Bookmark Middle-Click Activator)

Microsoft Edge（Chromium系ブラウザ）向けの拡張機能です。
ブックマークバーの項目を中クリック（ホイールクリック）で開いたとき、通常はバックグラウンドで開かれる新しいタブを自動的にアクティブ表示にします。

## 特長

- 新しいタブが作成されたときだけ動作する軽量設計（常駐キャッシュ・定期処理なし）
- ブックマークバー以外の通常のリンクの中クリック（バックグラウンドで開く動作）には影響しません
- 日本語UI対応（Chrome/Edge標準の `_locales` による多言語対応）

## 仕組み

ブックマークバーはブラウザ標準のUIのため、拡張機能から直接クリックイベントを検知することはできません。そのため本拡張機能は、「バックグラウンドで新しいタブが開かれ、かつそのURLがブックマーク登録済みのものと完全一致する」場合にのみ、そのタブを自動でアクティブ化する方式で目的の動作を実現しています。

## インストール方法

Microsoft Edgeストアでの公開は行っていないため、開発者モードでの導入が必要です。

1. このリポジトリを `Code > Download ZIP` からダウンロード、または `git clone` する
2. `edge://extensions` を開く
3. 左下の「開発者モード」をオンにする
4. 「展開して読み込み」（Load unpacked）をクリックし、`manifest.json` が直接入っているフォルダを選択する

## 権限について

| 権限 | 用途 |
|---|---|
| `tabs` | 新しく開かれたタブのURL取得・アクティブ化のため |
| `bookmarks` | 開かれたURLがブックマーク登録済みかどうかの判定のため |

外部への通信は一切行いません。
